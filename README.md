# Better Code Blocks

Plugin for markdown-it to have better code blocks

```terminal
npm i --save @jwevans/better-code-blocks
```

## Styles

```css
.named-code-block {
   // block styles
}

.named-code-block .named-code-block-title {
   // styles for file name
}

.named-code-block pre {
   // styles for code within block
}
```
